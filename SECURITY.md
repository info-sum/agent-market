# SECURITY POLICY (MVP)

## 1) 토큰/비밀정보
- OpenClaw 토큰 평문 저장 금지
- DB 저장 시 AES-256 암호화
- 서버 비밀키는 KMS 또는 환경변수 시크릿 관리
- 토큰 조회는 관리자도 원문 불가(마스킹)

## 2) 권한 최소화
- 연결 시 scope를 최소 권한으로 요청
- 설치/실행/파일수정 권한은 분리
- 민감 권한 변경 시 사용자 재동의 필요

## 3) 패키지 무결성
- 업로드 패키지 SHA-256 체크섬 검증
- 악성 패턴 자동 검사(스크립트/명령어 룰)
- 심사 미통과 버전은 배포 불가

## 4) 설치 안전성
- 설치 전 변경 파일 목록 고지
- 설치 전 스냅샷 생성 필수
- 실패 시 자동 롤백
- 수동 롤백 기능 제공

## 5) 감사/추적
- 주요 액션(연동, 결제, 설치, 승인/반려) audit log 기록
- 로그 최소 90일 보관
- 보안 사고시 타임라인 복원 가능해야 함
