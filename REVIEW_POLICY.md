# REVIEW POLICY (상품 심사 정책)

## 심사 대상
- Skill Pack
- MD Pack
- Persona Pack

## 심사 기준
1. 안전성: 악성 동작/권한 남용 여부
2. 명확성: 설명, 설치방법, 롤백 가이드
3. 재현성: 안내대로 설치/동작 가능한지
4. 품질: 오탈자/깨진 링크/누락 파일 여부

## 자동 검사 (Gate 1)
- 패키지 무결성
- 금지 패턴 탐지
- 메타데이터 필수값 확인

## 수동 심사 (Gate 2)
- 실제 샌드박스 설치 테스트
- 사용자 영향 범위 검토
- 정책 위반 여부 최종 판정

## 결과 상태
- Approved: 배포 가능
- Rejected: 반려 사유와 함께 수정 요청
- Needs Changes: 경미한 수정 후 재심사

## 반려 예시
- 과도한 권한 요구
- 설명/체인지로그 누락
- 외부 자산 라이선스 위반 가능성
- 설치 실패 재현

## SLA
- 1차 결과: 영업일 기준 2일 이내
