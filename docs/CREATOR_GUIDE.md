# CREATOR GUIDE (초안)

## 1. 무엇을 판매할 수 있나요?
- Skill Pack: 기능 실행 중심 패키지
- MD Pack: 운영 규칙/지침/프롬프트 정책
- Persona Pack: 캐릭터/톤/스타일 패키지

## 2. 등록 전 체크리스트
- [ ] 상품 설명 명확히 작성
- [ ] 대상 사용자/사용 조건 명시
- [ ] 설치 단계와 롤백 단계 문서화
- [ ] 버전/체인지로그 작성
- [ ] 필요한 권한(scope) 최소화

## 3. 권장 메타데이터
- 제목
- 카테고리
- 지원 OpenClaw 버전
- 권한 요구 목록
- 예상 효과(예: 응답 품질 향상, 작업시간 단축)

## 4. 가격 정책 팁
- 무료 체험판 + 유료 확장판 권장
- 초기 리뷰 확보를 위해 론칭 할인 활용
- 유지보수 범위를 설명에 반드시 명시

## 5. 업데이트 정책
- SemVer 권장 (MAJOR.MINOR.PATCH)
- 변경 사항은 반드시 changelog에 작성
- breaking change는 별도 경고

## 6. 금지 사항
- 악성 행위 또는 우회 스크립트 포함
- 사용자 정보 무단 수집
- 라이선스 위반 리소스 포함
