# Frontend Prototype

트렌디한 다크 그라디언트 UI의 정적 프로토타입(`index.html`)입니다.

## Run locally
```bash
cd frontend
python3 -m http.server 3000
# http://localhost:3000 접속
```

## 포함 기능
- JWT 인증 (회원가입/로그인)
- OpenClaw 연결 추가/목록
- 상품 조회/추가
- 설치 실행/이력
- 백엔드 상태/내 계정 상태 확인
